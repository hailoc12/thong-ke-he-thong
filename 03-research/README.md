# Research

Investigation and exploration work.

## How to Use
- Create subfolders for different research topics
- Store findings from web searches, documentation review
- Keep notes organized by theme or question being answered

## Structure
```
03-research/
├── [topic-1]/
│   ├── findings.md
│   └── sources.md
├── [topic-2]/
│   └── ...
└── README.md
```

## Note Format
```markdown
# Research: [Topic]

**Date**: [Date]
**Question**: [What are we trying to find out?]

## Sources
- [Source 1](url)
- [Source 2](url)

## Key Findings
1. Finding one
2. Finding two

## Implications
[How this affects our work]
```
