# Ideas

Brainstormed possibilities and potential solutions.

## How to Use
- Capture all ideas without judgment initially
- Evaluate and refine ideas over time
- Move promising ideas to solutions folders

## Note Format
```markdown
# Idea: [Idea Title]

**Status**: Raw | Evaluated | Promising | Rejected
**Date**: [Date]

## Description
[What is this idea?]

## Pros
- Pro 1
- Pro 2

## Cons
- Con 1
- Con 2

## Next Steps
[If promising, what to do next?]
```
