# Good Questions

Important questions that need answers.

## Status Tracking
Each question note MUST indicate:
- **Status**: `unanswered` | `answered`
- **Answer Reference**: Link to answer file if answered

## How to Use
- Capture unknowns and uncertainties
- Track which questions have been answered
- Link to answers in 09-good-answer/

## Note Format
```markdown
# Question: [Question Title]

## Status
- **Status**: unanswered
- **Answer Reference**: [[answer-file-name]] (when answered)
- **Last Updated**: [Date]

## The Question
[Full question with context]

## Why This Matters
[Why answering this is important]

## Potential Sources
- [Where might we find the answer?]
```

## Workflow
1. Create question with status `unanswered`
2. Research and create answer in `09-good-answer/`
3. Update this file: change status to `answered` and add reference
