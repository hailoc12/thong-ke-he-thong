# Resources

Reference materials, links, and assets.

## How to Use
- Store links to useful documentation
- Keep reference materials organized
- Include tools, libraries, and external resources

## Structure
```
07-resources/
├── links.md           # Useful URLs
├── tools.md           # Tools and software
├── documentation/     # Local copies of docs
└── assets/           # Images, files, etc.
```

## Note Format
```markdown
# Resource: [Resource Name]

**Type**: Link | Tool | Documentation | Asset
**URL**: [if applicable]

## Description
[What is this resource?]

## When to Use
[When is this resource helpful?]

## Notes
[Any important notes about using this]
```
