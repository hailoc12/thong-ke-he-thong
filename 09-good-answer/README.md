# Good Answers

Solutions and findings to questions from 06-good-question/.

## How to Use
- Create answers that reference original questions
- Include evidence and reasoning
- Always update the question file to mark as answered

## Note Format
```markdown
# Answer: [Answer Title]

## Answers Question
**Original Question**: [[question-file-name]]
**Action Required**: Update question file status to 'answered'

## The Answer
[Clear, complete answer]

## Evidence
- [Supporting evidence 1]
- [Supporting evidence 2]

## Sources
- [Source 1](url)
- [Source 2](url)

## Confidence Level
High | Medium | Low

## Notes
[Any caveats or additional context]
```

## Workflow
1. Create answer referencing original question
2. **Important**: Go back to question file and update status to `answered`
3. Add answer reference link in question file
