# AI

Tất cả thông tin liên quan đến sử dụng AI cho project này.

## Structure
```
17-AI/
├── prompts/           # Các prompt đã sử dụng
├── conversations/     # Lịch sử conversations quan trọng
├── outputs/           # AI-generated outputs cần lưu trữ
├── agents/            # Custom agent configurations
└── README.md
```

## How to Use

### Prompts
Lưu các prompt hiệu quả để tái sử dụng:
```markdown
# Prompt: [Tên prompt]

**Purpose**: [Mục đích]
**Model**: Claude | GPT-4 | Gemini
**Effectiveness**: High | Medium | Low

## Prompt
\`\`\`
[Nội dung prompt]
\`\`\`

## Example Output
[Ví dụ output tốt]

## Notes
[Ghi chú về cách sử dụng]
```

### Conversations
Lưu các conversations quan trọng:
```markdown
# Conversation: [Chủ đề]

**Date**: [Ngày]
**Model**: [Model sử dụng]
**Purpose**: [Mục đích]

## Summary
[Tóm tắt nội dung quan trọng]

## Key Insights
- Insight 1
- Insight 2

## Artifacts Generated
- [Link to file 1]
- [Link to file 2]
```

### AI Outputs
Lưu các outputs quan trọng từ AI:
- Code generated
- Analysis results
- Research summaries
- Design suggestions

### Agents
Cấu hình các custom agents cho project:
```markdown
# Agent: [Tên agent]

**Purpose**: [Mục đích chuyên biệt]
**Base Model**: [Model]

## System Prompt
\`\`\`
[System prompt cho agent]
\`\`\`

## Tools Available
- Tool 1
- Tool 2

## Usage Examples
[Ví dụ cách sử dụng agent]
```

## Best Practices

1. **Version prompts**: Đánh số version cho prompts quan trọng
2. **Tag by task**: Tag prompts theo loại task (research, coding, writing, etc.)
3. **Document failures**: Ghi lại prompts không hiệu quả và lý do
4. **Iterate**: Liên tục cải thiện prompts dựa trên kết quả
